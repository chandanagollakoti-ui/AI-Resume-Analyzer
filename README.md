# AI Resume Analyzer

AI-powered Resume Analyzer built using FastAPI, Streamlit, and Google Gemini AI.

## Features

- Upload PDF resumes
- Compare resumes with job descriptions
- AI-powered ATS-style analysis
- Resume strengths and weaknesses
- Improvement suggestions
- Download analysis reports

## Tech Stack

- FastAPI
- Streamlit
- Google Gemini AI
- Python
- PyPDF

## Run Backend

```bash
cd backend
uvicorn app:app --reload
```

## Run Frontend

```bash
cd frontend
streamlit run app.py
```
